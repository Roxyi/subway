# [Subway Ridership Analysis](https://roxyi.github.io/subway_ridership/)

![preview](assets/preview.jpg)

## Getting Started
Clone the repo
```
git clone https://github.com/Roxyi/subway_ridership.git
```

Install all dependencies
```
npm install
```

Run the app
```
Gulp
```
